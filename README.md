# Banking Transaction API

A Spring Boot REST API for managing banking operations including deposits, withdrawals, and fund transfers.

## Features

- REST APIs for account management
- Deposit and withdrawal operations
- Transaction-safe fund transfers
- Idempotent transfer APIs to prevent duplicate transactions
- Optimistic locking to handle concurrent balance updates

## Tech Stack

- Java
- Spring Boot
- PostgreSQL
- Spring Security
- Maven

## Key Concepts

- ACID transactions
- Idempotent API design
- Concurrency control with optimistic locking
